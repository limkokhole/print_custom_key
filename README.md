# print_custom_key

Print all gnome custom shortcut keys details. 

How to use:  

[1]  source it first, or normally put its code in ~/.bash_aliases  

$ chmod +x print_custom_key.sh; . print_custom_key.sh  

[2] Run as `print_custom_key`, enjoy :)
